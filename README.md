

# Quick Start

```sh
$ cd ~
$ git clone git@github.com:zhoukekestar/spotlight.git
$ cd spotlight
$ npm i
```

`Command` + `Space` 触发 spotlight，输入 `github cli` 获取自己的 github 名称。
